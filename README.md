# 📝 To-Do List App — Python Internship Project 1

A command-line To-Do List app built as Project 1 of my Python 
Programming Internship at DecodeLabs.

## About
Focuses on storing, adding, viewing, and removing items using Python 
lists — simulating how a lightweight in-memory database works.

## Features
- Add tasks
- View tasks with numbered indexing
- Remove tasks by number
- Input validation using `str.isdigit()` (no try/except)

## Concepts Covered
- Lists (`append()`, `pop()`)
- Loops & `enumerate()`
- Functions & `if __name__ == "__main__":`
- Input handling

## Run
```bash
python todo_list.py
```
